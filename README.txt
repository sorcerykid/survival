Basic Survival Mod
By Leslie E. Krause

Basic Survival is part of the Just Test Tribute game for the Minetest S3 engine.


Repository
----------------------

Browse source code...
  https://bitbucket.org/sorcerykid/survival

Download archive...
  https://bitbucket.org/sorcerykid/survival/get/master.zip
  https://bitbucket.org/sorcerykid/survival/get/master.tar.gz

Dependencies
----------------------

Default Mod (required)
  https://github.com/minetest/minetest_game

Configuration Panel Mod (required)
  https://bitbucket.org/sorcerykid/config

3d Armor Mod (required)
  https://bitbucket.org/sorcerykid/3d_armor

Timekeeper Mod (required)
  https://bitbucket.org/sorcerykid/timekeeper

Snowdrift II Mod (required)
  https://bitbucket.org/sorcerykid/snowdrift

Stopwatch Mod (required)
  https://bitbucket.org/sorcerykid/stopwatch
